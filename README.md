# Polytoria VR
Polytoria BepInEx mod that lets you play the game in full VR.
Mainly for use with [my custom game made for this mod](https://polytoria.com/places/98836),
however it can be used in any game, just without the hands.

BepInEx: https://github.com/BepInEx/BepInEx

### WARNING: Modding Polytoria is against the rules. *You might get banned from using this mod*. Prefer using an alt when the mod is active.

## Installation

1. Download the [latest release](https://github.com/cetotos/PolytoriaVR/releases/latest) from the releases tab, unzip the file and run the program, and click the "Install mod" button.
   
    This should automatically download and install BepInEx, BepInEx and the mod.


3. Install SteamVR (if you haven't already) and run it, make sure your headset is connected to SteamVR.
4. Enter any game and click "Activate VR" in the program. It should hopefully work!

If you encounter any bugs or issues, make a report in the [issues tab](https://github.com/cetotos/PolytoriaVR/issues).
