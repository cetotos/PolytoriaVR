# **Polytoria VR**
Polytoria BepInEx mod that lets you play the game in full VR.

BepInEx: https://github.com/BepInEx/BepInEx

### WARNING: Modding Polytoria is against the rules. You might get banned from using this mod. Prefer using an alt when the mod is active.

## Installation

Download the GUI from the releases tab, and click the "Install mod" button.

This should automatically download BepInEx and the mod.

Install SteamVR (if you haven't already), make sure your headset is connected to SteamVR, then enter a game and click "Activate VR".
