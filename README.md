# Polytoria VR
Polytoria BepInEx mod that lets you play the game in full VR.
Mainly for use with [my custom game made for this mod](https://polytoria.com/places/98836),
however it can be used in any game, just without the hands.

### WARNING: Modding Polytoria is against the rules. *You WILL get banned from using this mod*. USE AN ALT when the mod is active.

## Installation

You can watch the installation tutorial here (Thanks to CeeDee for tutorial):

https://www.youtube.com/watch?v=F0g_UpIyZZY

To delete the mod, search %appdata% in the windows search bar, go to the Polytoria folder and delete the folder called "Client".

The Polytoria launcher will automatically re-install the client.
